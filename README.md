# fedtorch
A federated optimization framework. The framework consists of the following methods: FedAVG, FedAVGM, DeltaSGD, DualPROX, FedEXP, FedGM, FedInit, FedLESAM, FedNAR, FedPROX, SCAFFOLD, SCAFFNEW, and FedADAM

    The running examples are located in the experiments directory. Simply configure the appropriate parameters in the libs file and run experiments/MNIST/run.py.
